# AD Privilege Escalation Analyzer

This tool analyzes Active Directory environments to identify privilege escalation risks and insecure configurations.

## Overview

The AD Privilege Escalation Analyzer identifies common Active Directory
misconfigurations that attackers can exploit to gain privileged access.

The tool performs automated enumeration of Active Directory objects
and highlights privilege escalation risks that are commonly abused
during post-compromise attack paths.

## Goals

- Identify nested privilege escalation paths
- Detect risky delegation configurations
- Identify stale privileged accounts
- Produce a security posture report

## Planned Checks

- Domain Admin inheritance chains
- Unconstrained delegation
- AdminCount accounts
- Privileged group membership
- Stale administrative accounts
